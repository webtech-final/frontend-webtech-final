# frontend-webtech-final

## Project setup

### Install dependence

```bash
yarn install
```

### Create .env file

```bash
cp .env.example .env
```

### Edit .env file with text editer

```
VUE_APP_ENDPOINT={host ip}
VUE_APP_WEBSOCKET_ENDPOINT={host ip}
```

### Run FrontEnd server

```bash
yarn serve
```

### RUN AT PORT 8080
go to http://localhost:8080/
